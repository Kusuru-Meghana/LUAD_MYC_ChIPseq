# Peaks (MACS2 Output)

This folder contains the primary peak-calling results generated from MYC ChIP-seq analysis using **MACS2**.

### Files included
- **MYC_A549_peaks.narrowPeak** – Main peak file with coordinates and peak metrics.
- **MYC_A549_peaks.xls** – Detailed peak table including enrichment scores.
- **MYC_A549_summits.bed** – Summits representing the most enriched binding point within each peak.
- **MYC_A549_model.r** – MACS2 predicted fragment size model.

These files represent genome-wide MYC binding sites in A549 (LUAD) cells.
