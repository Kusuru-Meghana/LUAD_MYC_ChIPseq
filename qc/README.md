# Quality Control (FastQC + MultiQC)

This folder contains quality control results for raw ChIP-seq reads.

### Contents
- FastQC reports for each FASTQ file
- MultiQC summary report (aggregated QC metrics)

These QC metrics confirm:
- per-base read quality
- GC content
- adapter contamination levels
- duplication levels

All samples passed QC and were used for downstream alignment.
